# Section 6: Custom Shortcut Recommendations

## Recommended Browser Extensions & Custom Shortcuts

### 1. Vimium (Browser Extension)
Transform your browser into a keyboard-driven powerhouse:
- **f** - Click any link with letter hints
- **F** - Open link in new tab
- **/** - Search page
- **j/k** - Scroll down/up
- **gg/G** - Go to top/bottom
- **yy** - Copy current URL

### 2. Custom Bookmarklets for AI
Create these as bookmarks for one-click access:

```
javascript:(function(){window.open('https://chat.openai.com','_blank')})()
```
**Name:** Open ChatGPT

```
javascript:(function(){window.open('https://claude.ai','_blank')})()
```
**Name:** Open Claude

### 3. AutoHotkey Scripts (Windows)
Create custom shortcuts for frequent AI tasks:

```autohotkey
; Win + G opens ChatGPT
#g::Run, https://chat.openai.com

; Win + C opens Claude
#c::Run, https://claude.ai

; Ctrl + Alt + P pastes clipboard and presses Enter
^!p::
Send, ^v
Sleep, 100
Send, {Enter}
return
```

### 4. Mac Automator Shortcuts
Create Quick Actions bound to keyboard shortcuts:

**AI Research Flow:**
1. Open Automator > Quick Action
2. Add "Run Shell Script"
3. Script: `open -a "Google Chrome" "https://claude.ai"`
4. Save as "Open Claude"
5. System Preferences > Keyboard > Shortcuts > Services
6. Assign: Cmd + Shift + A

### 5. Text Expansion for Prompts
Set up text expansion (TextExpander, Espanso, or AutoHotkey):

| Trigger | Expands To |
|---------|------------|
| `;explain` | "Explain this concept in simple terms: " |
| `;improve` | "Please improve this text for clarity and professionalism: " |
| `;code` | "Write clean, well-commented code that: " |
| `;summ` | "Summarize the key points from the following: " |
| `;email` | "Draft a professional email that: " |

# Section 7: Power User Workflows

## Workflow 1: Research Sprint

1. **Ctrl + Shift + N** - New browser window (incognito for clean results)
2. **Ctrl + T** (x3) - Open tabs for ChatGPT, Claude, Perplexity
3. **Ctrl + L** - Focus address bar, type URL
4. **Ctrl + 1/2/3** - Quick switch between tabs
5. **Ctrl + Shift + C** - Copy AI responses
6. **Ctrl + Shift + V** - Paste without formatting to notes

## Workflow 2: Content Creation Pipeline

1. **Research Phase:**
   - Use Perplexity with **Ctrl + Shift + F** (Focus mode)
   - Copy sources with **Ctrl + Shift + C**

2. **Draft Phase:**
   - Open Claude, paste context
   - Use **Shift + Enter** for multi-line prompts
   - **Escape** to stop if off-track

3. **Refinement Phase:**
   - Copy draft to new ChatGPT chat
   - Iterate with "improve" prompts
   - Use text expansion for common refinements

## Workflow 3: Code Development with AI

1. **Ctrl + Shift + I** - Open DevTools alongside AI
2. Ask AI for code, **Ctrl + Shift + ;** - Copy code block
3. Paste to IDE, test, copy errors
4. **Shift + Enter** for multi-line error paste
5. Iterate until resolved
