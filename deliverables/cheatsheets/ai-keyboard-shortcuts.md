## Midjourney Shortcuts (Discord-Based)

### Command Shortcuts
| Command | Function | Pro Tip |
|---------|----------|---------|
| `/imagine` | Generate image | Add `--v 6` for latest version |
| `/describe` | Get prompts from image | Upload image first |
| `/blend` | Merge 2-5 images | Great for style mixing |
| `/info` | Check subscription status | Shows fast hours remaining |
| `/settings` | Open settings panel | Toggle modes quickly |
| `/prefer suffix` | Set default parameters | e.g., `--ar 16:9 --style raw` |
| `/prefer option set` | Create custom shortcuts | Save complex parameter combos |

### Parameter Quick Reference
| Parameter | Function | Example |
|-----------|----------|---------|
| `--ar` | Aspect ratio | `--ar 16:9`, `--ar 9:16`, `--ar 1:1` |
| `--v` | Version | `--v 6`, `--v 5.2` |
| `--style raw` | Less stylized | More photorealistic |
| `--q` | Quality | `--q 2` (higher = more detail) |
| `--c` | Chaos | `--c 50` (more variety) |
| `--s` | Stylize | `--s 750` (more artistic) |
| `--no` | Negative prompt | `--no text, watermark` |
| `--seed` | Reproducibility | `--seed 12345` |
| `--tile` | Seamless patterns | Great for backgrounds |
| `--repeat` | Multiple versions | `--repeat 4` (up to 40) |

### Button Actions (After Generation)
| Button | Function |
|--------|----------|
| `U1-U4` | Upscale selected image |
| `V1-V4` | Create variations |
| `Re-roll` | Regenerate all 4 |
| `Vary (Strong)` | Major variations |
| `Vary (Subtle)` | Minor tweaks |
| `Zoom Out 2x/1.5x` | Expand canvas |
| `Pan arrows` | Extend in direction |

## Pro Tips & Workflow Hacks

### Speed Optimization Combos

#### The "Rapid Fire" Workflow (ChatGPT/Claude)
1. `Cmd/Ctrl + Shift + O` - New chat
2. Type prompt + `Enter` - Submit
3. `Cmd/Ctrl + Shift + C` - Copy response
4. `Cmd/Ctrl + Tab` - Switch apps
5. `Cmd/Ctrl + V` - Paste

#### The "Power User" Image Gen (Midjourney)
1. `/prefer suffix --ar 16:9 --v 6 --style raw` - Set defaults once
2. `/imagine [prompt]` - Generate (defaults auto-applied)
3. Use `V1-V4` for quick variations
4. `Vary (Subtle)` for refinement

#### The "Code Assistant" Flow (Copilot/Cursor)
1. Type comment describing function
2. `Tab` - Accept suggestion
3. `Cmd/Ctrl + K` - Inline edit for tweaks
4. `/fix` - Auto-fix errors
5. `/tests` - Generate tests

### Custom Shortcut Setup

#### Chrome Extension: Shortkeys
- Create custom shortcuts for any AI tool
- Map complex actions to single keys
- Sync across devices

#### AutoHotkey (Windows) / Keyboard Maestro (Mac)
- Create macros for repetitive prompts
- Auto-paste common prompt starters
- Chain multiple actions together

### Clipboard Manager Integration
- **Windows**: Ditto (`Ctrl + ~`)
- **Mac**: Raycast (`Cmd + Shift + V`)
- Store frequently used prompts
- Quick access to 50+ clipboard items

### Multi-Tool Workflow Example

**Task: Generate Marketing Assets**
1. Claude (`Cmd + Shift + O`): Generate copy
2. `Cmd + Shift + C`: Copy response
3. Midjourney: `/imagine [paste concept]`
4. Gamma: Generate presentation with both

**Time saved: ~70% vs manual workflows**
