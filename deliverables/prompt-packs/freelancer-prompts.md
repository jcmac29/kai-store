## Finding & Landing Clients (8 Prompts)

### 1. Client Persona Builder
```
Help me define my ideal client.

My freelance service: [what you offer]
Current clients: [who you've worked with]
Best projects: [which ones went well]
Worst projects: [which ones were difficult]
What I enjoy: [work I like doing]
Income goal: [monthly target]

Create 2-3 ideal client personas:
For each:
- Company type/industry
- Company size
- Their role (who hires you)
- Budget range
- Problems they have
- How they find freelancers
- What matters to them
- Red flags in this persona
- Where to find them online
```

### 2. Proposal Template Builder
```
Create a proposal template for my services.

Service: [what you offer]
Client type: [who you serve]
Typical project: [common scope]
My pricing: [rates/packages]

Build a proposal with:
1. Personalized intro (showing I understand them)
2. Problem statement (reflect their situation)
3. Proposed solution (what I'll do)
4. Scope of work (deliverables list)
5. Timeline/milestones
6. Pricing (with options if applicable)
7. Why me (brief credibility)
8. Next steps (clear CTA)
9. Terms (payment, revisions)

Keep it to 1-2 pages max. Professional but warm.
```

### 3. Cold Outreach Templates
```
Write cold outreach messages for finding clients.

My service: [what you offer]
Target client: [who you're reaching]
Platform: [email/LinkedIn/Twitter/DM]
My unique angle: [what makes you different]

Create outreach for these situations:
1. Reaching out to a company with an obvious problem
2. Reaching out after seeing a job posting
3. Reaching out to someone in my network
4. Reaching out after they posted content
5. Follow-up when no response

Each message:
- Under 100 words
- Personal (not template-y)
- Clear value proposition
- Soft CTA
- Easy to reply to
```

### 5. Portfolio Case Study Writer
```
Write a portfolio case study.

Client: [who - can anonymize]
Project: [what you did]
Challenge: [problem they faced]
Solution: [what you delivered]
Results: [outcomes achieved]
Testimonial: [if available]

1. Client overview (industry, size)

```

## Client Management (8 Prompts)

### 9. Contract Template Builder
```
Create a freelance contract template.

Service: [what you offer]
Typical project scope: [common deliverables]
Payment structure: [hourly/project/retainer]
My concerns: [what issues to protect against]

Include clauses for:
1. Scope of work
2. Payment terms
3. Timeline/deadlines
4. Revision policy
5. Kill fee/cancellation
6. Intellectual property
7. Confidentiality
8. Liability limits
9. Termination conditions
10. Dispute resolution

Note: I'm not a lawyer. This is a starting template to review with legal counsel.
```

### 10. Scope Creep Deflector
```
Help me handle scope creep.

Current project: [what the original scope was]
The creep: [what they're asking for now]
Client relationship: [good/tense/new]
My preference: [accommodate / push back / charge more]

Draft responses for:
1. Acknowledging the request
2. Explaining it's outside scope
3. Offering options:
   - Add to scope (with cost)
   - Save for phase 2
   - Swap for something else
4. Maintaining relationship

Firm but professional. Not passive-aggressive.
```
