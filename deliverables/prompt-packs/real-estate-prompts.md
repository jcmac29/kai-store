### 1. Compelling Listing Description Generator
```

- Address: [address]
- Property type: [single family, condo, townhouse, etc.]
- Bedrooms: [number]
- Bathrooms: [number]
- Square footage: [sq ft]
- Lot size: [if applicable]
- Year built: [year]
- Key features: [list 5-10 notable features]
- Recent upgrades: [any renovations]
- Neighborhood highlights: [schools, parks, shopping, etc.]

Write a description that:
1. Opens with an attention-grabbing hook
2. Highlights the property's best features
3. Creates emotional appeal (lifestyle, not just specs)
4. Includes neighborhood benefits
5. Ends with a call to action

Provide 3 versions:
- Full MLS description (250-300 words)
- Social media version (100 words)
- Email teaser (50 words)
```
**What it does:** Creates professional, engaging listing descriptions optimized for different platforms.
