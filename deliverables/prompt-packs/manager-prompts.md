## 1:1 Meeting Prompts (6)

### 1. Weekly 1:1 Agenda Builder
```
Help me create an effective 1:1 agenda.

Team member: [their name and role]
Tenure: [how long they've been in role]
Current projects: [what they're working on]
Recent context: [wins, challenges, changes]
My goals for this 1:1: [what I want to accomplish]
Meeting length: [time available]

Create an agenda with:
1. Check-in opener (rapport building)
2. Their topics first (what they want to discuss)
3. Progress check (without micromanaging)
4. Obstacle removal (how I can help)
5. Development discussion (growth focus)
6. Alignment items (things I need to share)
7. Action items and next steps

Also provide:
- 3 open-ended questions for each section
- Time allocation suggestions
- Topics to avoid or postpone
- Follow-up actions to take
```
