## Twitter/X (10 Prompts)

### 1. Viral Tweet Generator
```
Generate viral tweets about [topic].

Topic: [your subject]
Audience: [who follows you]
Goal: [engagement/reach/DMs]
Your voice: [how you sound]

Generate 10 tweets using these formats:
1. Hot take / controversial opinion
2. Observation (I noticed...)
3. Lesson learned (Here's what I learned...)
4. Challenge (Most people think X, but actually...)
5. List (3 things...)
6. Question that gets replies
7. Story hook (Thread incoming:)
8. Direct value (How to...)
9. Bold prediction
10. Personal win/loss

Each under 280 characters. Punchy.
```

### 2. Twitter Thread Writer
```
Write a Twitter thread about [topic].

Topic: [your subject]
Thread goal: [educate/tell story/promote]
Thread length: [7-15 tweets]
Tone: [casual/authoritative/personal]
CTA: [what at the end]

Structure:
Tweet 1: Hook (must stop scroll)
Tweets 2-X: Content (one idea each)
Last tweet: CTA + recap

Format each tweet:
- Number (1/10)
- Under 280 chars
- No threads that could be a blog post

First tweet is everything. Nail the hook.
```

### 3. Quote Tweet Templates
```
Create quote tweet templates.

Account type: [personal brand/business/meme]
Usual topics: [what I talk about]

Templates for QRT-ing:
1. Adding value to someone's point
2. Respectfully disagreeing
3. Sharing personal experience
4. Asking follow-up question
5. Highlighting key point
6. Adding controversy/spice
7. Making a joke
8. Supporting/amplifying
9. Connecting to my expertise
10. Turning into a thread starter

Short, punchy, makes them click expand.
```

### 4. Twitter Bio Optimizer
```
Optimize my Twitter bio.

What I do: [your thing]
Who I help: [audience]
Proof: [credentials/achievements]
Personality: [vibe]
CTA: [if any]

Create 5 bio options:
- Under 160 characters each
- Clear value proposition
- Personality showing
- Social proof included
- Keywords for search

Include: Location, link recommendations, pinned tweet idea.
```

### 5. Engagement Bait (Ethical)
```
Create engagement posts that aren't cringe.

My niche: [topic area]
Audience: [who follows]
What I want: [replies/RTs/followers]

Generate 10 engagement posts:
1. "Unpopular opinion: [take]"
2. Fill in the blank
3. This or that choice
4. "What's your [X]?"
5. Ranking challenge
6. Hot take that gets agreement
7. Confession format
8. "Nobody talks about [X]"
9. Prediction ask
10. Advice request

Make people WANT to reply, not feel obligated.
```

## Instagram (10 Prompts)

### 6. Carousel Content Planner
```
Create an Instagram carousel.

Topic: [subject]
Slides: [8-10]
Goal: [save/share/follow]
Audience: [who sees it]

For each slide provide:
- Headline (bold, 6 words max)
- Supporting text (1-2 lines)
- Visual suggestion

Structure:
Slide 1: Hook (stop scroll)
Slides 2-8: Value delivery
Slide 9: Summary
Slide 10: CTA (save this / follow for more)

Make each slide valuable alone but better together.
```

### 7. Instagram Caption Formula
```
Write Instagram captions.

Post type: [single image/carousel/Reel]
Topic: [what it's about]
Tone: [vibe]
CTA: [what you want them to do]

Write 3 versions:
1. Short (under 100 chars)
2. Medium (story + value)
3. Long (deep dive)

Include:
- Hook first line
- Line breaks for readability
- Emoji usage (if appropriate)
- Hashtag set (20-30)
- CTA

First line is the only thing they see. Make it count.
```

### 8. Reels Script Writer
```
Write an Instagram Reel script.

Topic: [what the Reel is about]
Length: [15/30/60 seconds]
Format: [talking head/b-roll/trending]
Goal: [views/follows/saves]
Hook type: [question/bold claim/curiosity]

Script format:
HOOK (0-3 sec): [stop the scroll]
SETUP (3-8 sec): [context]
CONTENT (8-X sec): [the meat]
CTA (last 3 sec): [what to do]

Include:
- Text overlay copy
- Sound/music suggestion
- Visual transitions
- Caption
```

### 9. Story Sequence Planner
```
Plan an Instagram Story sequence.

Purpose: [engage/sell/share/connect]
Total stories: [5-10]
End goal: [link click/DM/poll response]
My voice: [how I sound]

Create sequence:
Story 1: Hook/attention grab
Story 2-X: Content/journey
Final: CTA

For each story:
- Visual (photo/video/text graphic)
- Text overlay
- Stickers to use (poll, question, emoji slider)
- Music if applicable

Keep momentum through all stories.
```

### 10. Hashtag Strategy
```
Create a hashtag strategy for my account.

Niche: [your area]
Account size: [followers]
Content types: [what you post]
Goal: [reach/engagement/community]

Create hashtag sets:
1. Big hashtags (1M+ posts) - 5 tags
2. Medium hashtags (100K-1M) - 10 tags
3. Small hashtags (10K-100K) - 10 tags
4. Niche hashtags (under 10K) - 5 tags

Plus:
- Content-specific tags
- When to use which
- Hashtags to avoid
- Community hashtags to engage with
```

## LinkedIn (10 Prompts)

### 11. LinkedIn Post Formatter
```
Write a LinkedIn post.

Topic: [what I'm sharing]
Post type: [story/lesson/opinion/tip]
My role: [what you do]
Audience: [who follows you]
Goal: [engagement/DMs/awareness]

Write using LinkedIn best practices:
- Strong opening line (hook)
- Short paragraphs (1-2 lines)
- White space between thoughts
- Personal perspective
- Specific examples (not generic)
- CTA at end

One idea per post. Deep, not wide.

## TikTok / Short-Form (10 Prompts)

### 16. TikTok Hook Writer
```
Write TikTok hooks (first 2 seconds).

Content topic: [what videos are about]
Target audience: [who watches]
Video style: [talking head/voiceover/trending]

Generate 20 hooks:
1. Question hooks ("Did you know...")
2. Challenge hooks ("I bet you can't...")
3. Story hooks ("So this happened...")
4. Controversial hooks ("Unpopular opinion...")
5. Tutorial hooks ("How to... in 30 seconds")
6. Result hooks ("I tried X and...")
7. Secret hooks ("The secret to...")
8. Mistake hooks ("Stop doing this...")
9. List hooks ("3 ways to...")
10. Direct hooks ("You need to hear this...")

Pattern interrupt in 2 seconds or you're dead.
```

### 17. TikTok Script Outliner
```

Style: [educational/funny/story/rant]

Script structure:
HOOK (0-3 sec): [stop scroll, create curiosity]
SETUP (3-7 sec): [context]
BODY (7-X sec): [the value/story]
PAYOFF/CTA (end): [resolution + action]

Include:
- Exact words to say
- Pause/emphasis marks
- B-roll/cutaway suggestions
- Text overlay copy
- Sound/music recommendation
- Caption for post
```

```

## Platform-Specific Tips

**Twitter/X:** Hooks + hot takes + threads

**LinkedIn:** Stories + insights + professionalism
**TikTok:** First 2 seconds + trends + watch time

---

## How to Use This Pack

1. **Pick your platform** - Focus on one at a time
2. **Choose your goal** - Engagement? Followers? Sales?
3. **Find the prompt** - Match to your need
4. **Fill in brackets** - Be specific
5. **Post and iterate** - See what works

**Pro tip:** Screenshot your best performing content. Use it as reference in future prompts.
