## Video Scripts (10 Prompts)

### 1. YouTube Long-Form Script
```
Write a YouTube video script on [topic].

Video length: [5-15 minutes]
Channel niche: [your niche]
Target audience: [who watches]
Goal: [educate/entertain/convince]

Include:
1. Hook (first 30 seconds) - pattern interrupt, make them stay
2. Intro - establish credibility, preview value
3. Main content - 3-5 key points with examples
4. Story or case study - make it memorable
5. Call to action - subscribe, comment, etc.
6. Outro - tease next video

Tone: [conversational/professional/energetic]
Add notes for b-roll and visual suggestions.
```

### 2. YouTube Shorts/TikTok Script
```
Write a viral short-form video script.

Topic: [what's the video about]
Hook type: [question/shock/curiosity/controversy]
Length: [30-60 seconds]
Platform: [TikTok/Shorts/Reels]

Structure:
1. Hook (2-3 seconds) - stop the scroll
2. Setup (5-10 seconds) - context
3. Value (20-40 seconds) - the meat
4. Payoff/CTA (5 seconds) - end strong

Write the exact words to say. Make it punchy.
Include notes for: cuts, text overlays, sounds.
```

### 3. Educational Video Script
```
Create a tutorial video script.

Topic: [what you're teaching]
Skill level: [beginner/intermediate/advanced]
Video length: [target duration]
Format: [screen recording/talking head/mixed]

Structure:
1. Problem statement (what we're solving)
2. What you'll learn (3-4 things)
3. Prerequisites (if any)
4. Step-by-step walkthrough
5. Pro tips/common mistakes
6. Recap and next steps

Write conversationally. Include pauses for demonstrations.
```

### 4. Podcast Episode Outline
```
Create a podcast episode outline.

Topic: [episode subject]
Episode type: [solo/interview/co-host]
Length: [target duration]
Audience: [who listens]

Generate:
1. Episode title (catchy, searchable)
2. Episode description (2-3 sentences)
3. Intro (hook + what's covered)
4. Main segments (3-4 distinct sections)
5. Talking points for each segment
6. Questions to explore
7. Stories/examples to share
8. Outro (takeaway + CTA)
9. Timestamps outline
```

### 5. Course Module Script
```
Write a script for a course module.

Course topic: [overall course]
Module topic: [this specific lesson]
Module number: [where in sequence]
Length: [target duration]
Student level: [who's taking the course]

Include:
1. Module intro (why this matters)
2. Learning objectives (3-4 things)
3. Concept explanations
4. Examples/demonstrations
5. Practice prompts for students
6. Common questions addressed
7. Key takeaways
8. What's next preview

Keep it engaging. Don't be boring.
```

### 6. Livestream Run Sheet
```
Create a livestream run sheet.

Stream topic: [what you're doing]
Platform: [Twitch/YouTube/Instagram]
Duration: [planned length]
Stream type: [just chatting/gaming/tutorial/AMA]

Generate:
1. Pre-stream checklist
2. Intro segment (first 5 min)
3. Main content segments with times
4. Engagement moments (polls, Q&A)
5. Transition phrases
6. Emergency topics (if chat is slow)
7. Raid/host targets
8. Outro + next stream preview

Include call-outs for chat interaction.
```

### 7. Product Review Video Script
```
Write a product review video script.

Product: [what you're reviewing]
Category: [tech/beauty/fitness/etc.]
My honest opinion: [positive/mixed/negative]
Video length: [target]
Affiliate link: [yes/no]

Structure:
1. Hook (show the product, make a claim)
2. Quick verdict (for impatient viewers)
3. Unboxing/first impressions
4. Features deep dive
5. Pros (detailed)
6. Cons (be honest)
7. Comparison to alternatives
8. Who this is for / not for
9. Final verdict + rating
10. Where to buy (if affiliate)

Be genuine. People smell fake reviews.
```

### 8. Story Time Video Script
```
Write a story time video script.

Story premise: [what happened - brief]
Emotion: [funny/scary/cringe/wholesome]
Length: [target duration]
Platform: [YouTube/TikTok]

Structure:
1. Hook (tease the best part)
2. Context (set the scene)
3. Rising action (build tension)
4. Climax (the main event)
5. Resolution (what happened after)
6. Reflection (what I learned/thought)
7. CTA (ask viewers about their stories)

Write exactly how you'd tell it out loud.
Include pauses, expressions, emphasis notes.
```

### 9. Comparison Video Script
```
Write a versus/comparison video script.

Items being compared: [A vs B]
Category: [what type of products]
My recommendation: [A/B/depends]
Video length: [target]

Structure:
1. Hook (state the comparison, tease winner)
2. Overview of both options
3. Criteria we're judging on
4. Category-by-category comparison:
   - [Criterion 1]: A vs B
   - [Criterion 2]: A vs B
   - [Criterion 3]: A vs B
   - etc.
5. Price comparison
6. Summary table
7. Final verdict with reasoning
8. Who should pick A / who should pick B
```

Help me structure my vlog.

Mood: [chill/exciting/emotional]

2. Intro + context

4. Segment 2: [second thing]
5. B-roll break (what to show)

Add notes for: music vibes, pacing, transition ideas.
```

## Hooks & Headlines (10 Prompts)

### 11. Viral Hook Generator
```
Generate 10 viral hooks for my video.

Topic: [what the video is about]
Platform: [YouTube/TikTok/IG]
Emotion to trigger: [curiosity/shock/FOMO/outrage]
Target audience: [who]

Create hooks using these formulas:
1. Bold claim
2. Controversial opinion
3. Question that demands an answer
4. "You're doing X wrong"
5. Number + result ("I made $X doing...")
6. Story tease ("What happened next...")
7. Us vs them
8. Pattern interrupt
9. Direct call out ("Hey [audience]")
10. Fear of missing out

Make each under 10 words. Punchy.
```

### 12. Title A/B Test Generator
```
Generate 10 title variations for A/B testing.

Video topic: [what's it about]
Current title: [if you have one]
Target keyword: [for SEO, if any]
Platform: [YouTube/article/etc.]

Create titles using:
- Numbers (7 ways, 10 tips)
- How to
- Questions
- Superlatives (best, worst, most)
- Year (2026)
- Brackets [FREE] [UPDATED]
- Emotional words
- Curiosity gaps

Rank them by clickability. Note which are best for SEO.
```

### 13. Thumbnail Text Ideas
```
Suggest thumbnail text for my video.

Video title: [your title]
Video topic: [what's it about]
Emotion to convey: [shock/excitement/etc.]

Generate:
- 5 options with 1-3 words (best for thumbnails)
- 5 options with 4-5 words (more context)

For each, note:
- Color contrast suggestion
- Expression suggestion (for face)
- Emoji or icon ideas (if any)

Less is more. Make it readable on mobile.
```

### 14. Pattern Interrupt Hooks
```
Write 10 pattern interrupts for the first 3 seconds.

Content type: [vlog/tutorial/talking head]
Topic: [what the video is about]
Usual opening in my niche: [what others do]

Create pattern interrupts that:
- Break expectations
- Create immediate curiosity
- Feel different than everything else
- Work without context

Examples to model:
- Starting mid-sentence
- Sound effect + visual
- Wrong emotion (laugh about something sad setup)
- Direct challenge to viewer
```

### 15. Call-to-Action Templates
```
Write CTAs for different goals.

My content niche: [what you make]
My typical audience: [who watches]
Platform: [where you post]

Create CTAs for:
1. Subscribe/follow
2. Turn on notifications
3. Comment engagement
4. Like the video
5. Share with a friend
6. Check out another video
7. Join membership/Patreon
8. Buy my product/course
9. Use my affiliate link
10. Follow on other platform

Make each feel natural, not desperate.
Two versions: subtle and direct.
```

## Social Posts & Captions (10 Prompts)

### 16. Twitter/X Thread Writer
```
Write a Twitter thread about [topic].

Thread length: [5-15 tweets]
Goal: [educate/entertain/promote]
Tone: [casual/professional/spicy]
CTA: [what you want at the end]

Format each tweet:
- Under 280 characters
- One idea per tweet
- Hook in tweet 1
- Number tweets (1/10, 2/10...)
- End with CTA

First tweet must stop the scroll.
Last tweet should be self-contained (for QRTs).
```

### 17. Instagram Caption Formula
```
Write an Instagram caption.

Post type: [carousel/single/Reel]
Topic: [what the post is about]
Goal: [engagement/reach/DMs/sales]
Tone: [your vibe]
Include CTA: [yes/no, what kind]

Structure:
1. Hook (first line is everything)
2. Value or story (middle section)
3. CTA (what to do)
4. Hashtag suggestions (10-20)

Two versions:
- Short (under 150 characters)
- Long (storytelling style)
```

### 18. LinkedIn Post Formatter
```
Write a LinkedIn post about [topic].

Post type: [story/opinion/tip/announcement]
Industry: [your field]
Goal: [thought leadership/engagement/leads]
Tone: [professional but not boring]

Format:
- Strong opening line (hook)
- Single-line paragraphs
- Strategic white space
- Personal perspective
- Specific examples
- Engaging question at end
- 3-5 relevant hashtags

Avoid: corporate jargon, humble brags, obvious engagement bait.
```

### 19. Newsletter Intro Writer
```
Write a newsletter introduction.

Newsletter name: [what you call it]
Topic this week: [what you're covering]
Tone: [casual/professional/witty]
Subscriber relationship: [new/established]

Create:
1. Subject line options (3)
2. Preview text
3. Opening section that:
   - Greets naturally (not "Hey [NAME]!")
   - Hooks with curiosity
   - Sets up what's coming
   - Feels like a friend writing

Make it skimmable. Bold key phrases.
```

### 20. Engagement Question Generator
```
Generate engagement questions for my niche.

Content niche: [what you create]
Platform: [where you post]
Audience: [who follows you]
Goal: [comments/saves/shares/DMs]

Create 20 questions in these categories:
- Opinion questions (hot takes)
- "This or that" questions
- Story prompts ("Tell me about...")
- Advice requests ("What would you do if...")
- Nostalgia prompts ("Remember when...")
- Confession prompts ("Unpopular opinion...")
- Help requests ("I need your input...")
- Challenge questions ("Can you...")

Make them impossible to scroll past without responding.
```

## Content Planning (5 Prompts)

### 21. Content Pillar Strategy
```
Create content pillars for my brand.

Niche: [what you create content about]
Platform: [primary platform]
Target audience: [who you want to reach]
Business goal: [what content should achieve]

Define 4-5 content pillars:
For each pillar:
- Name it (simple, memorable)
- What it covers
- Why audience cares
- Content formats it works for
- Posting frequency
- Example topics (5-10 each)

Then create a weekly content mix using the pillars.
```

### 22. 30-Day Content Calendar
```
Create a 30-day content calendar.

Niche: [what you create]
Platforms: [where you post]
Posting frequency: [daily/X times per week]
Content types: [what formats you create]
Upcoming events: [holidays, launches, etc.]

For each day, provide:
- Content type
- Topic/idea
- Platform
- Draft hook/caption
- Any ties to trends/events

Include:
- Evergreen content
- Trending topics
- Personal content
- Promotional content (20% max)
```

### 23. Content Repurposing Plan
```
Help me repurpose this content.

Original content: [describe it]
Original platform: [where it was posted]
Performance: [how it did]
Other platforms I'm on: [list them]

Create a repurposing plan:
1. Extract key points/moments
2. Adapt for each platform
3. Suggest formats (carousel, thread, clip)
4. Write new hooks for each
5. Identify best times to post
6. Note what to change vs keep

Maximize reach without creating from scratch.
```

### 24. Trending Topic Adapter
```
Help me create content around this trend.

Trend: [what's trending]
Platform: [where you want to post]
My niche: [how to connect it]
My take: [your angle]

Create:
1. Why this trend matters to my audience
2. My unique perspective
3. 3 content ideas using this trend
4. Scripts/outlines for each
5. How to stay relevant without being cringe
6. Timing recommendation (now vs wait)

Help me ride the wave without being a try-hard.
```

### 25. Series Concept Developer
```
Help me create a content series.

Niche: [your content area]
Platform: [where you'll post]
Format: [video/carousel/thread]
Goal: [build audience/promote product/establish expertise]

Develop a series concept:
1. Series name (catchy, memorable)
2. Core premise (what it's about)
3. Episode structure (repeatable format)
4. First 10 episode topics
5. Intro/outro templates
6. Visual branding suggestions
7. Hashtag/naming convention
8. Posting schedule
9. How to promote the series
10. When to end/evolve it
```

## Bonus Prompts (5)

### 26. Viral Content Analyzer
```
Analyze why this content went viral.

Content: [link or describe it]
Platform: [where it performed]
Metrics: [views, shares, etc.]

Break down:
1. The hook - why people stopped
2. The format - why it worked
3. The topic - why people cared
4. The timing - why now
5. The emotion - what it triggered
6. The shareability - why people spread it
7. What I can learn from it
8. How to apply it to my content (without copying)
```

### 27. Audience Avatar Builder
```
Help me understand my target audience.

My content niche: [what you create]
Current audience: [who follows now]
Dream audience: [who you want]
Platform: [primary platform]

Create detailed avatars (2-3 people):
For each:
- Name, age, situation
- What they struggle with
- What they want to achieve
- Where they hang out online
- What content they consume
- What makes them follow someone
- What makes them unfollow
- How to earn their trust
- What triggers them to share
- How my content helps them
```

### 28. Collab Pitch Template
```
Write collaboration pitch messages.

My niche: [what I create]
My audience size: [rough numbers]
My unique value: [what I bring]
Target creator type: [who to collab with]

Create pitches for:
1. Similar-sized creator
2. Bigger creator (reaching up)
3. Brand partnership
4. Podcast guest pitch
5. Guest post/takeover pitch

Each should:
- Be under 100 words
- Lead with value for them
- Be specific (not generic)

- Feel human, not template-y
