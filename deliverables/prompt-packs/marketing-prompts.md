## Copywriting (10 Prompts)

### 1. Sales Page Writer
```
Write a sales page for my product.

Product: [what you're selling]
Price: [amount]
Target customer: [who buys this]
Main problem it solves: [their pain]
Key benefits: [3-5 benefits]
Features: [list features]
Proof: [testimonials, results, credentials]

Write a full sales page with:
1. Headline (stop them, make a promise)
2. Subheadline (clarify the value)
3. Opening hook (agitate the problem)
4. Story/case study (show transformation)
5. Feature/benefit breakdown
6. Objection handling
7. Social proof section
8. Pricing section
9. Guarantee
10. Multiple CTAs
11. FAQ section
12. Final push

Make it persuasive but not sleazy.
```

### 2. Email Sales Sequence
```
Write a sales email sequence.

Product: [what you're selling]
Price: [amount]
Audience: [who's on the list]
Launch type: [new launch/evergreen/limited]
Sequence length: [5-7 emails]

For each email:
- Subject line (+ 2 alternatives)
- Preview text
- Full email body
- CTA

Email purposes:
1. Problem awareness
2. Solution introduction
3. Proof/case study
4. Objection handling
5. Urgency/scarcity
6. Final call

Include timing recommendations.
```

### 3. Landing Page Copy
```
Write landing page copy.

Page purpose: [lead capture/sales/signup]
Target visitor: [who lands here]
Traffic source: [where they come from]
Offer: [what they get]
Goal action: [what you want them to do]

Write:
1. Hero section (headline, subhead, CTA)
2. Problem section
3. Solution section
4. Benefits (bullet points)
5. How it works (3 steps)
6. Social proof
7. FAQ (5 questions)
8. Final CTA section

Keep it scannable. Above the fold = most important.
```

### 4. Product Description Writer
```

Target buyer: [who buys]
```

### 10. Headline Formula Generator
```
Generate headlines using proven formulas.

Topic: [what you're promoting]
Audience: [who you're targeting]
Benefit: [main value]
Unique mechanism: [how it works]

Generate 5 headlines for each formula:

1. How to [benefit] without [pain]
2. [Number] ways to [desired outcome]
3. The [adjective] [thing] that [benefit]
4. Why [thing] doesn't work (and what does)
5. [Audience]: Here's how to [benefit]
6. Stop [mistake]. Do [this] instead.
7. What [authority] knows about [topic]
8. The truth about [topic]
9. [Result] in [timeframe]
10. Are you making these [topic] mistakes?
```

## How to Use This Pack

1. **Find your need** - Skim categories for your current challenge
2. **Fill in details** - The more specific, the better output
3. **Iterate** - First draft → feedback → refinement
4. **Test** - Run A/B tests on copy variations
5. **Track** - Measure what works for YOUR audience

**Pro tip:** Keep a swipe file of your best-performing copy. Feed it back into prompts for even better results.
